:v: Zumbleflick
:house: robotyczny ślimak

W wiosce {v} mieszkała mała wynalazczyni o imieniu Plonka Nibbertwist. Plonka zbudowała {house}, który pędził szybciej niż gepard! Pewnego dnia ścigał się z wiatrem - i wygrał. Od tego dnia Plonka i jej ślimak jeździli dookoła, pomagając każdemu, kto potrzebował uśmiechu albo kanapki.
